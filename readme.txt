Este plugin para insertar de forma dinámica CTAs en un blog

== Description ==
 Este plugin para insertar de forma dinámica CTAs en un blog, la idea es que para que todo tenga sentido se pueda insertar el CTA adecuado al contenido, tipo de producto y servicio y buyer journey, esperamos que sea de utilidad.

== Installation ==
1.Visita la página de plugins en tu escritorio y selecciona «Añadir nuevo»;
2.Busca «Sistemex Inbound»;
3.Activa Sistemex Inbound desde tu página de plugins;
